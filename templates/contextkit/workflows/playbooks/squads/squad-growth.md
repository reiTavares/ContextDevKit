# 📈 Playbook: growth-team

This playbook details discovery loops, search engines indexability, and user cohort telemetry.

## 👥 Members
* `growth`: Oversees activation metrics, onboarding funnels, and feature loops.
* `retention`: Audits cohort churn factors, lifecycle communications, and messaging.
* `seo-specialist`: Assesses page sitemaps, indexability, and answer engine visibility.

## 📝 Best Practices
1. **Event Telemetry:** When publishing a new flow, ensure conversion tracking logs are registered in analytics (aligned with privacy limits).
2. **SEO Auditing:** Keep routes indexable. Check meta titles, sitemaps, and robots tags.
3. **No Dark Patterns:** Avoid misleading interface loops to optimize short-term metrics. Preserve user trust.
